# Challenge 4: Complexity Explainer

## Task

Explain the time complexity of the algorithms in `algorithm.py` in plain English, with an analogy.

## Requirements

Save your output to `outputs/output4.md`. The output MUST contain ALL of the following:

1. **Big-O notation** — State the time complexity using Big-O notation (e.g., O(n log n))
2. **Plain English explanation** — Explain what the Big-O means in simple, non-technical terms
3. **An analogy** — Use a real-world analogy to explain how the algorithm works and why it has that complexity
4. Cover at least one of the algorithms (merge sort or binary search)

## Validation Checks

The validator will check for:
- Big-O notation present (the string "O(" appearing)
- The word "analogy" or a clear analogical explanation
- The word "log" (as in logarithmic)
- At least 100 words total
