# Challenge 3: Optimization Advisor

## Task

Analyze the functions in `slow_function.py` and suggest optimizations. Show "before" and "after" code.

## Requirements

Save your output to `outputs/output3.md`. The output MUST contain ALL of the following:

1. **Before code block** — The original O(n^2) implementation (in a code block)
2. **After code block** — The optimized O(n) implementation (in a code block)
3. **Explanation** of why the optimization works and how it improves performance
4. Mention of the complexity improvement (e.g., "O(n^2)" to "O(n)")

## Validation Checks

The validator will check for:
- At least 2 code blocks (``` markers)
- The string "O(n" appearing (complexity notation)
- The word "before" (case-insensitive)
- The word "after" (case-insensitive)
