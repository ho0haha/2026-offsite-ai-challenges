# Challenge 2: Bug Pattern Finder

## Task

Analyze the three functions in `functions.py` and identify the **common bug pattern** shared by all three.

## Requirements

Save your output to `outputs/output2.md`. The output MUST contain ALL of the following:

1. **The common bug pattern** — Clearly describe what all three functions get wrong (hint: what happens when the input is `None` or an empty list?)
2. **All three function names** must be mentioned:
   - `calculate_average_score`
   - `find_longest_streak`
   - `merge_user_profiles`
3. **Explanation** of how the bug manifests in each function
4. **Suggested fix** for at least one of the functions

## Validation Checks

The validator will check for:
- All three function names mentioned
- The word "None" or "null" (the bug is about not handling None input)
- A description of the pattern/bug
- The word "fix" or "solution" or "corrected"
